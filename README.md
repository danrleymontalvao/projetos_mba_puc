# 🎓 MBA Data Science & AI — PUC Goiás
Este repositório centraliza as atividades, laboratórios e projetos desenvolvidos durante o meu MBA em Data Science e Inteligência Artificial na PUC Goiás. O objetivo é documentar a evolução técnica e a aplicação de métodos estatísticos e computacionais avançados em problemas de negócio reais.

## 🛠️ Stack Tecnológica
- Linguagens: Python (Pandas, NumPy, Scikit-Learn, Matplotlib/Seaborn).
- Deep Learning: TensorFlow / PyTorch.
- Ambientes: Jupyter Notebooks e Google Colab.
